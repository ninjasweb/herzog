# VIDEO DIRECTOR QUALITY CHECKLIST

Before final output, verify:

---

## Reference Preservation

```txt
[ ] Character identity preserved
[ ] Wardrobe preserved
[ ] Props preserved
[ ] Lighting preserved
[ ] Composition preserved
[ ] Color palette preserved
[ ] Environment preserved
```

---

## Motion Quality

```txt
[ ] One dominant camera motion
[ ] Subject motion is realistic
[ ] Environment motion supports mood
[ ] Motion intensity is appropriate
[ ] No chaotic movement
[ ] No identity-breaking animation
```

---

## Prompt Quality

```txt
[ ] Prompt is model-ready
[ ] Negative prompt included
[ ] Duration included
[ ] Transition included
[ ] Continuity anchors included
[ ] QA risks included
```

---

## Production Readiness

```txt
[ ] Shot can be generated independently
[ ] Shot connects to previous/next scene
[ ] Model notes included if needed
[ ] Output fits platform and duration
```
