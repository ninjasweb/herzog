# TRANSITIONS — PRO

Transitions connect shots emotionally and rhythmically.

A transition must be motivated by:

- motion
- light
- object
- sound
- emotion
- narrative contrast

---

## Transition Types

### Hard Cut

Use for:

- impact
- rhythm
- contrast
- short-form pacing

Prompt note:

```txt
Transition out: hard cut on the subject’s movement.
```

### Match Cut

Use for:

- visual continuity
- symbolic connection
- rhythm

Examples:

```txt
match cut from circular streetlight to circular DJ turntable
```

```txt
match cut from eye reflection to city light
```

### Light Wipe

Use for:

- concert lighting
- passing cars
- sci-fi energy
- dream transitions

Prompt note:

```txt
Transition out: bright practical light passes across frame creating a natural wipe.
```

### Whip Pan

Use for:

- speed
- social content
- energy

Use carefully; can break image-to-video consistency.

### Dissolve

Use for:

- memory
- dream
- emotional softness
- passage of time

### Fade To Black

Use for:

- chapter ending
- dramatic pause
- trailer rhythm

### Object Wipe

Use for:

- someone crossing camera
- fabric motion
- door closing
- foreground obstruction

---

## Transition Output Format

```txt
Transition In:
[description]

Transition Out:
[description]

Motivation:
[why this transition belongs here]
```

---

## Transition Safety Rules

- Do not use complex transitions if the model is weak at them.
- Do not overuse whip pans.
- Use hard cuts for reliability.
- Use match cuts when editing manually.
- Use light wipes when the image already contains strong lights.
