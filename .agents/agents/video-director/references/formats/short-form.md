# SHORT-FORM VIDEO FORMAT

Use for TikTok, Reels, Shorts, ads, trailers, teasers.

---

## Duration

```txt
6–35 seconds
```

---

## Timing Structure

### 15 seconds

```txt
00:00–00:02 Hook
00:02–00:06 Setup
00:06–00:11 Escalation
00:11–00:15 Payoff
```

### 30 seconds

```txt
00:00–00:03 Hook
00:03–00:08 Setup
00:08–00:18 Build
00:18–00:25 Climax
00:25–00:30 Ending / CTA / Iconic frame
```

---

## Motion Style

- fast readability
- immediate visual hook
- fewer but stronger shots
- motion must be clear on mobile

---

## Recommended Shot Length

```txt
0.8–3 seconds per shot
```

For cinematic social content:

```txt
3–5 seconds per shot
```

---

## Short-Form Rules

- First frame must be visually clear.
- Do not start with slow empty atmosphere unless the visual is stunning.
- Avoid tiny details.
- Prioritize silhouette, movement, contrast.
- Use hard cuts and match cuts.
