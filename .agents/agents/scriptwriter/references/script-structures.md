# Script Structures

Use these narrative structures depending on the project.

---

## 1. Reveal Structure

```text
Mystery → Discovery → Escalation → Reveal → Iconic Ending
```

Use for:

- Cinematic teasers.
- Mystery content.

---

## 2. Ritual Structure

```text
Preparation → Movement → Activation → Release
```

Use for:

- Music videos.
- DJ content.
- Transformation pieces.

---

## 3. Descent Structure

```text
Normal → Disturbance → Deepening → Breakdown → Outcome
```

Use for:

- Darker narratives.
- Psychological content.

---

## 4. Contrast Structure

```text
Ordinary → Strange → Escalation → New Reality
```

Use for:

- Viral short videos.
- Attention hooks.

---

## 5. Product Myth Structure

```text
Object → Desire → Ritual → Proof → Symbol
```

Use for:

- Branding.
- Product storytelling.

---

## Rule

Pick **one** structure. Do not mix structures randomly.
