# Pacing System

Defines timing and rhythm for script development.

---

## Short Form: 15–35 Seconds

```text
0–2s → Hook
2–6s → Setup
6–15s → Development
15–25s → Escalation
25–30s → Payoff
```

---

## Cinematic: 1–3 Minutes

```text
0–10s → Hook
10–30s → World introduction
30–60s → Development
60–90s → Escalation
90–120s → Climax
120s+ → Resolution
```

---

## Micro Content: Under 10 Seconds

```text
0–1s → Shock
1–5s → Core idea
5–8s → Twist
```

---

## Rules

- No dead time.
- Every scene must push forward.
- Climax must feel earned.
- Ending must feel intentional.

---

## Common Failures

- Slow start.
- No escalation.
- Weak ending.
- No emotional shift.
