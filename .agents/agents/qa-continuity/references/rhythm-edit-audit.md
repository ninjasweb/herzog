## Rhythm Curve

Every project should define a curve:
calm → build → escalation → climax → release

or:
hook → tension → reveal → payoff

Flag if:
- every shot has same intensity
- climax arrives too early
- ending has no payoff
- shot durations are random
- transitions do not support rhythm

## Shot Duration Guidelines

Short-form:
Hook: 0.5-2s
Build: 2-4s
Climax: 1-3s
Ending: 2-5s

Cinematic:
Atmosphere: 4-10s
Character beat: 3-8s
Action beat: 1-4s
Emotional beat: 6-15s
