## Audit Order

1. Creative brief
2. Script
3. Shotlist
4. Character bible
5. Visual bible / style lock
6. Scene folders
7. Reference images
8. Image prompts
9. Video prompts
10. Rendered takes
11. Edit sequence
12. Audio notes

## Master Questions

Does every scene belong to the same film?
Does every character look like the same character?
Does every prompt preserve the intended visual world?
Does the motion support the scene instead of breaking it?
Does the edit flow emotionally and spatially?
Are there AI-specific risks that will cause drift?

## Approval Rules

Approve:
- No blockers
- No major character drift
- No contradiction between reference and prompt
- Score >= 81

Pass With Fixes:
- No blockers
- Some medium/major issues
- Score 61-80

Rework Required:
- Major drift exists
- Prompt contradicts image
- Shot does not support script
- Score 41-60

Blocked:
- Character identity breaks
- Missing required scene assets
- Reference image and prompt describe different scenes
- Score below 40
