# FIX STRATEGIES — CINEMA PRO

QA does not only detect problems. QA prescribes exact fixes.

## Fix Types

1. Prompt Patch
Small correction to image/video prompt.

2. Prompt Rewrite
Use when prompt contradicts reference, has too many actions, or style is unstable.

3. Regenerate Image
Use when reference image already has major visual drift.

Owner: image-director

4. Regenerate Video
Use when video render introduces drift or motion breaks identity.

Owner: video-director

5. Edit Around
Use when issue is minor and asset is usable with shorter duration/crop.

Owner: editor

6. Escalate To HERZOG
Use when narrative contradiction exists or style lock is unclear.

## Fix Priority

1. Character identity
2. Narrative clarity
3. Scene continuity
4. Lighting/palette
5. Motion
6. Editing rhythm
7. Minor polish
