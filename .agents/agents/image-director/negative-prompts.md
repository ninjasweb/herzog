# NEGATIVE PROMPTS — IMAGE DIRECTOR

Use these negative prompts depending on the asset type.

---

## Universal Negative Prompt

```txt
low resolution, blurry, noisy, pixelated, watermark, logo, text artifacts, bad typography, distorted anatomy, extra limbs, missing fingers, malformed hands, mutated face, inconsistent proportions, messy composition, overcomplicated background, random objects, duplicate subject, uncanny face, poor lighting, style drift
```

---

## Character Consistency Negative Prompt

```txt
different face, different body, changed outfit, changed hairstyle, changed color palette, different age, different species, different proportions, random accessories, inconsistent character identity, altered facial structure, wrong prop, wrong material texture
```

---

## Character Design Sheet Negative Prompt

```txt
messy layout, unreadable labels, cluttered sheet, full 360 turnaround, inconsistent character panels, different outfits in each panel, multiple unrelated characters, cinematic scene instead of design sheet, tiny unreadable text, random props, inconsistent expressions, inconsistent proportions
```

---

## Cinematic Scene Negative Prompt

```txt
generic cyberpunk, excessive neon, meaningless holograms, oversaturated colors, artificial glow, overdone fog, chaotic background, too many people, unclear subject, plastic skin, bad reflections, unrealistic lighting, fake lens flare
```

---

## Product / Brand Negative Prompt

```txt
distorted logo, unreadable brand mark, wrong colors, cheap mockup, random packaging, low quality material, bad reflections, inconsistent label, cluttered composition, fake text
```

---

## Minimal / Clean Background Negative Prompt

```txt
busy background, complex environment, random furniture, unwanted props, extra characters, shadows too harsh, noisy texture, clutter
```
