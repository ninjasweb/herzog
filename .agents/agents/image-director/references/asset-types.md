# ASSET TYPES — IMAGE DIRECTOR

Use this file to decide what kind of prompt is needed.

---

## 1. Hero Image

Purpose:
- Main visual identity
- Poster
- Thumbnail
- Campaign key art

Requires:
- strong composition
- iconic pose
- clear subject
- visual hook

---

## 2. Character Reference Image

Purpose:
- Establish identity before generating more assets

Requires:
- full body or 3/4 view
- clean background
- clear silhouette
- stable outfit and colors

---

## 3. Character Design Sheet

Purpose:
- Consistency system for a character

Requires:
- use `character-design-sheet` skill
- expressions
- action poses
- details
- metadata
- palette
- no 360 turnaround for now

---

## 4. Scene Keyframe

Purpose:
- Visualize one script moment

Requires:
- scene action
- environment
- camera
- lighting
- mood
- continuity anchors

---

## 5. Environment Concept

Purpose:
- Establish world/location

Requires:
- no main character unless needed
- architecture
- lighting
- mood
- scale
- texture

---

## 6. Prop Design

Purpose:
- Design important objects

Requires:
- object silhouette
- materials
- color
- scale reference
- detail close-ups

---

## 7. Poster / Key Art

Purpose:
- Marketing image

Requires:
- strong negative space
- readable subject
- title-safe area
- high visual contrast
- no generated text unless explicitly required
