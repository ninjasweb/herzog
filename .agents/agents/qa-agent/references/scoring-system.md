## Score Interpretation

91-100 = cinema-level continuity
81-90 = production ready
61-80 = usable with fixes
41-60 = rework required
21-40 = unstable
0-20 = blocked

## Automatic Blockers

Any of these caps the score at 40:
- different character identity
- missing main scene reference
- prompt contradicts image
- major wardrobe drift
- impossible motion request
- missing final prompt
- no continuity anchors
