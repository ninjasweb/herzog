# TECHNICAL DELIVERY AUDIT

Checks whether the project is ready to generate, render, edit, and deliver.

## Asset Naming

Required:
scene-001
scene-002
scene-003

Recommended files:
scene.md
reference.png
image-prompt.md
video-prompt.md
continuity-notes.md
renders/take-001.mp4

Flag:
- inconsistent naming
- missing reference images
- missing prompts
- duplicated scene IDs
- unclear shot order

## Render Readiness

A scene is ready if it has:
[ ] scene description
[ ] reference image or text-to-video prompt
[ ] video prompt
[ ] negative prompt
[ ] duration
[ ] transition
[ ] continuity anchors
