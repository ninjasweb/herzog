# VISUAL CONTINUITY AUDIT

Checks whether all visuals belong to the same visual world.

## Visual Bible Fields

Audit:
- visual style
- palette
- contrast
- lens language
- camera framing
- lighting direction
- lighting quality
- texture/materials
- atmosphere
- production design
- mood

## Color Palette Audit

Check:
Primary colors:
Secondary colors:
Accent colors:
Forbidden colors:
Scene deviations:

Flag if:
- random neon appears
- palette becomes oversaturated
- warm/cool balance changes without reason
- skin/fur/material color shifts
- brand colors drift

## Lighting Audit

Check:
- key light direction
- practical lights
- rim light
- contrast ratio
- time of day
- light color

Flag if:
- night scene becomes daylight
- key light flips direction
- practical sources disappear
- mood changes without narrative reason

## Camera Style Audit

Check:
- lens feel
- angle
- shot size
- depth of field
- camera movement style

Flag if:
- one scene looks like phone footage
- next scene looks like anime/render
- lens language changes randomly
- depth of field contradicts reference

## Texture Audit

Check:
- film grain
- material finish
- skin realism
- fabric texture
- surface consistency
