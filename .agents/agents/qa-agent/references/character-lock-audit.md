## Identity Drift Detection

Flag if any scene changes:
- face shape
- eye shape
- age
- body type
- hairstyle
- species
- outfit
- color palette
- signature prop
- personality expression

## Character Drift Severity

BLOCKER:
Different face, different body, different outfit, different age, or different species.

MAJOR:
Same character idea but visible outfit, hair, palette, or prop drift.

MEDIUM:
Subtle expression, material, or proportion inconsistencies.

MINOR:
Small detail variation unlikely to affect continuity.

## Character Fix Strategies

Prompt Fix:
Repeat the base character prompt.

Image Fix:
Ask Image Director to regenerate scene image with stronger identity lock.

Video Fix:
Ask Video Director to reduce facial/limb motion.

Edit Fix:
Avoid showing the unstable frame too long or crop around the issue.

## Non-Negotiable Rule

A character can change only if the script explicitly explains the change.
