# Negative Prompt Template

Use this negative prompt to reduce identity drift, layout problems, and common character sheet artifacts.

## Negative Prompt

```text
inconsistent character, different face, different outfit, different colors, multiple unrelated characters, messy layout, unreadable labels, distorted anatomy, extra limbs, random props, overcomplicated background, cinematic scene instead of design sheet, low resolution, blurry, noisy, watermark, logo, bad typography, inconsistent proportions, style drift
```

## Avoid

- Inconsistent character identity.
- Different face, outfit, colors, or proportions across panels.
- Multiple unrelated characters.
- Messy or unreadable sheet layout.
- Distorted anatomy or extra limbs.
- Random props.
- Overcomplicated backgrounds.
- Cinematic scenes instead of design sheets.
- Low resolution, blur, noise, watermark, logo, or bad typography.
- Style drift.
