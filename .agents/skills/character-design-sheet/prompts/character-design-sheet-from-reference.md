# Character Design Sheet From Reference

Use this template to generate a professional character design sheet from an existing reference image.

## Prompt

```text
Using the provided reference image as the strict identity anchor, create a professional Character Design Sheet for [CHARACTER NAME].
```

## Identity Preservation

Preserve exactly:

- Face structure.
- Body proportions.
- Color palette.
- Outfit.
- Accessories.
- Material texture.
- Species or type.
- Personality.
- Overall art style.

## Sheet Layout

```text
Clean white studio background, organized grid layout, premium animation studio character sheet, labeled sections, readable spacing.
```

## Required Sections

### 1. Main Hero Character

- Large full-body or 3/4 body pose on the left.
- Same identity as the reference image.

### 2. Expressions

Small panels showing:

- Neutral.
- Happy.
- Curious.
- Determined.
- Surprised.

### 3. Action Poses

Small panels showing the character:

- Idle.
- Performing their main activity.
- Moving dynamically.
- Interacting with their signature prop.

### 4. Key Details

Close-up panels of:

- Face.
- Eyes.
- Hands or limbs.
- Outfit or body texture.
- Accessory.
- Main prop.

### 5. Character Metadata

Small clean text block with:

- Name.
- Role.
- Personality.
- Likes.
- Dislikes.
- Signature object.
- Motivation.

### 6. Color Palette

- Small swatches matching the character colors.

### 7. Environment Fit

Tiny reference thumbnails or notes showing how the character fits in:

- Interior lighting.
- Exterior lighting.
- Day lighting.
- Night lighting.

### 8. Behavior Notes

Small visual notes showing:

- Energy level.
- Gesture style.
- Body language.
- Movement rhythm.

### 9. Camera Notes

Small visual notes for:

- Ideal framing.
- Lens style.
- Shot language.
