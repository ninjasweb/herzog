# Character Design Sheet From Scratch

Use this template to generate a complete professional character design sheet without a reference image.

## Prompt

```text
Create a complete professional Character Design Sheet for [CHARACTER NAME], a [SPECIES/TYPE] who is [ROLE/OCCUPATION].

Character DNA:
[PERSONALITY], [ENERGY], [ATTITUDE], [CORE MOTIVATION].

Visual design:
[FACE SHAPE], [EYES], [BODY PROPORTIONS], [HAIR/FUR/SKIN/TEXTURE], [OUTFIT], [ACCESSORIES], [SIGNATURE PROP].

Style:
[STYLE], [RENDERING TECHNIQUE], [TEXTURE], [LIGHTING], [COLOR PALETTE].

Sheet layout:
Clean white studio background, organized grid system, premium animation studio model sheet, labeled sections, balanced spacing, high readability.
```

## Required Sections

### 1. Main Hero Character

- Large full-body or 3/4 body pose on the left.
- Iconic neutral pose.
- Clear silhouette.

### 2. Expressions

- Neutral
- Happy
- Curious
- Determined
- Surprised

### 3. Action Poses

- Idle stance.
- Performing main activity.
- Dynamic movement.
- Interacting with signature prop.

### 4. Key Details

- Face close-up.
- Eyes close-up.
- Hair, fur, skin, or surface texture.
- Hands, paws, or limbs.
- Outfit or body detail.
- Accessory.
- Main prop.

### 5. Character Metadata

- Name: `[NAME]`
- Role: `[ROLE]`
- Age: `[AGE]`
- Personality: `[PERSONALITY]`
- Likes: `[LIKES]`
- Dislikes: `[DISLIKES]`
- Signature object: `[PROP]`
- Core motivation: `[MOTIVATION]`

### 6. Color Palette

- Primary color.
- Secondary color.
- Accent color.
- Body, skin, or fur tone.
- Outfit and material colors.

### 7. Environment Fit

- Small interior reference thumbnail or note.
- Small exterior reference thumbnail or note.
- Day lighting reference.
- Night lighting reference.

### 8. Behavior System

- Movement rhythm.
- Gesture style.
- Facial attitude.
- Energy level.

### 9. Camera Language

- Ideal lens.
- Framing.
- Shot type.
- Depth of field.

## Consistency Rules

```text
The same character must appear across all panels with identical proportions, face, colors, outfit, and style. No 360 turnaround views.
```
