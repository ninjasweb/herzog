# HERZOG DIRECTOR OUTPUT

## 1. Creative Brief

Video homenaje realista, estilo pelicula, para Paloma: una french poodle blanca, pequena, de pelaje rizado y orejas suaves, vista saludable, alegre y llena de vida. La pieza debe sentirse como un reencuentro amoroso, no como una despedida triste.

Tono: luminoso, tierno, esperanzador.

Estilo: realismo cinematografico, luz dorada, nubes suaves, arcoiris natural, atmosfera celestial calida.

## 2. Selected Agents

- `character-designer`: fija la identidad visual de Paloma.
- `scriptwriter`: estructura el video de 15 segundos.
- `image-director`: crea prompts listos para imagen/video.

## 3. Execution Plan

```text
00:00-00:03
Primer plano de Paloma en un campo dorado. Se ve sana, feliz, con su pelaje blanco rizado, orejas suaves y expresion alegre. Mira hacia una luz calida en el cielo.

00:03-00:07
Paloma empieza a correr con energia. La camara baja la sigue desde el frente y luego desde un lateral. El pasto se mueve suavemente, aparecen particulas de luz.

00:07-00:11
Un arcoiris nace entre nubes luminosas. Paloma corre sobre el, ligera y feliz, como si el camino fuera natural para ella.

00:11-00:15
Llega a una nube calida donde la esperan otros perritos y familiares queridos en siluetas suaves, luminosas y respetuosas. Paloma se une a ellos y mira hacia camara con alegria.
```

## 4. Output Structure

This project should maintain separate outputs for character identity, script timing, and image/video prompts.

## 5. Next Steps

Generate a still keyframe of Paloma in the meadow, validate likeness against the user reference, then animate the strongest still.
